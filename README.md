# Personal Assistant

A Python-based terminal personal assistant application.

This project is an ongoing application designed to manage personal information and daily tasks through a command-line interface.

## Current Features

* User registration
* User login
* Personal notes management
* Add notes
* View notes
* Edit notes
* Delete notes
* Separate data storage for each user using JSON files

## Technologies

* Python 3
* JSON
* prompt-toolkit
